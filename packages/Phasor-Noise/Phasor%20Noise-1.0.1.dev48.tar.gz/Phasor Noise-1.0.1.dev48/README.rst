About The Project
------------------
We created a Phasor noise generator with a graphical interface to visualize how the different parameters affect the noise created.

Installation
-------------

1. Install from PyPI

.. code:: console

   pip install phasor_noise

Usage
---------
You can now use our Phasor noise generator.
